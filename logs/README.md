# logs/

Esta carpeta contiene salidas generadas en ejecución.

## Reglas
- `trades.csv`: SOLO operaciones reales cerradas.
- `decisions.log`: decisiones internas (para diagnóstico).
- `errors.log`: errores y excepciones.

> Por defecto, los logs no se suben a GitHub (están en .gitignore).
> Se mantiene el folder con `.gitkeep`.
